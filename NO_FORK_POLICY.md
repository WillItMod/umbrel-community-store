# No Fork Policy

This repository is source-available for transparency and internal use under the
Business Source License. It is not an open-source repo and it is not intended
to be forked or republished.

## Rule

You may not create, publish, maintain, or redistribute:

- Public forks of this repository
- Public mirrors of this repository
- Derivative repositories based on this repository
- Repackaged stores, releases, images, binaries, or bundles derived from this
  repository

unless you have explicit written permission from the Licensor.

## What is still allowed

- Reading and auditing the repository
- Using it internally as allowed by `LICENSE`
- Making private internal modifications for your own use

## Why this exists

The repository is published for transparency and for official release or store
distribution, not for community forks or competing redistributions.

See `LICENSE`, `LICENSE_POLICY.md`, and `COMMERCIAL_LICENSE.md` for the
governing terms.
